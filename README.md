# Novalife Plugin MyDelorean
Ce plugin Novalife MyDelorean offre la possibilité d'activé le voyage temporel ! (ou presque) 

## Table des Matières

- [Installation](#installation)
- [Utilisation](#utilisation)
- [Droits de Propriété Intellectuelle](#droits-de-propriété-intellectuelle)

## Installation

1. Téléchargez le fichier `MyDelorean.dll` depuis la page des releases de ce dépôt. 
2. Ajoutez le fichier `MyDelorean.dll` dans le dossier des plugins de votre serveur Novalife.

## Utilisation

1. Utiliser la commande /mydelorean
2. Montez dans une Delorean CMD-12 BTTF (ID: 33)
3. Rouler à plus de 88 km/h en direction d'un mur (ou pas)
